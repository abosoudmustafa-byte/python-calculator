# 🧮 Python Calculator

A simple command-line calculator built with Python.

## Features
- Addition ➕
- Subtraction ➖
- Multiplication ✖️
- Division ➗
- Division by zero protection 🛡️

## How to Run
```bash
python calculator.py
```

## Usage
1. Enter first number
2. Enter second number
3. Choose operation (1/2/3/4)
4. See the result!
